Project Description

Spots is a visually engaging website that demonstrates how to create a responsive grid-based layout for showcasing images and text content.
The design focuses on accessibility, readability, and flexibility, ensuring the interface remains consistent across devices.

The main objectives of this project were:

To master the use of Flexbox and CSS Grid for layout.
To implement a responsive design using media queries.
To organize the project using the BEM (Block Element Modifier) methodology.
To create a pixel-perfect version of a Figma design.
Features
Fully responsive layout for mobile, tablet, and desktop.
Semantic HTML structure for better accessibility.
Consistent design based on a provided Figma mockup.
Clean file organization following BEM conventions.
Optimized image assets for faster loading.
